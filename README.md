# cloudwatch-consumer-lambda

That AWS Lambda function listen CloudWatch log group(s) for new logs and transfer them to ElasticSearch.

### Requirements:
 - Java 11
 - Maven 3.6
 
 ### How to build:
 ```
mvn package
```
